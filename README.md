# 🍇 Winery Information System

A full-stack web application built with PHP and MySQL designed to digitize and streamline winery operations. This system manages everything from vineyard maintenance and grape harvesting to wine production and direct e-commerce sales.

## 🚀 Key Features

* **Multi-Role System:** Custom dashboards and permission levels for Administrators, Winemakers, Workers, Registered Customers, and Visitors.
* **Vineyard Management:** Track individual vineyard rows, grape varieties, planting years, and plan agricultural treatments (spraying, fertilizing, pruning).
* **Harvest & Production Logging:** Record harvest metrics (weight, sugar content) and track the wine production lifecycle (vintage, alcohol percentage, bottling dates).
* **Direct B2C E-commerce:** A built-in storefront allowing customers to browse the wine catalog, make purchases, and view their order history.
* **Business Logic Enforcement:** Automated checks, such as preventing chemical treatments within 14 days of a planned harvest.
