Implementace

Informačný systém je implementovaný ako webová aplikácia v jazyku PHP (verzia 8.4) s využitím frameworku Laravel. Architektúra systému vychádza z návrhového vzoru MVC. Dáta sú ukladané v relačnej databáze MySQL. Frontend využíva šablónovací systém Blade a CSS framework Bootstrap. Na implementáciu datepickeru sme použili AJAX JavaScript.

Prípady použitia na skripty

Jednotlivé funkčné celky a prípady použitia sú implementované v nasledujúcich kontroleroch a modeloch:

  • Autentifikácia a autorizácia: Registrácia, prihlásenie a odhlásenie užívateľov sú riešené v app/Http/Controllers/AuthController.php. Riadenie prístupových práv (Roles/Permissions) je zabezpečené pomocou middleware a policies (app/Policies/).
  • Správa užívateľov (Admin): CRUD operácie nad užívateľmi, prideľovanie rolí a správa účtov sú implementované v app/Http/Controllers/UserController.php.
  • Správa vinohradov: Evidencia riadkov vinohradu, odrôd a ich vlastností je v app/Http/Controllers/WineyardRowController.php.
  • Zber úrody (Harvests): Zadávanie a správa zberov, kontrola cukornatosti a dátumov zberu sú obsluhované v app/Http/Controllers/HarvestController.php.
  • Ošetrovanie a postreky: Plánovanie a evidencia vykonaných ošetrení vo vinohrade je riešená v app/Http/Controllers/TreatmentController.php.
  • Výroba vína (Šarže): Proces fľašovania (vytvorenie vínnej šarže z dokončeného zberu) a evidencia skladových zásob je implementovaná v app/Http/Controllers/WineBatchController.php.
  • Predaj a objednávky (E-shop): Zákaznícka sekcia pre nákup vína a prezeranie histórie objednávok je spravovaná v app/Http/Controllers/PurchaseController.php.
  • Dashboard a prehľady: Zobrazovanie súhrnných informácií pre rôzne role (Admin, Vinár, Zákazník) zabezpečuje app/Http/Controllers/DashboardController.php.

Štruktúra pohľadov (Views)

Užívateľské rozhranie je rozdelené do adresárov v resources/views/ podľa funkčných modulov (napr. harvests/, wine_batches/, users/). Hlavný layout aplikácie je definovaný v resources/views/layouts/app.blade.php.

Databáze

Instalace

Postup instalace na server:
  • composer install
  • touch .env
  • cp .env.example .env
  • php artisan key:generate
  • vložiť kľúč do premennej APP_KEY v súbore .env
  • npm install
  • npm run dev # pre vývoj
  • npm run build # pre produkciu
Softwarové požadavky:
  • PHP verzia 8.4
  • sqlite3 3.45.1
Jak rozbalit, konfigurovat, inicializovat databázi:
  • touch database/database.sqlite
  • php artisan migrate --seed
  • php artisan serve
  • localhost:8000/

Známé problémy

  • Neboli sme rozhodnutí ako implementovať plánované dátumy oproti skutočným dátumom vykonaných činností vo vinohrade. Z tohto dôvodu sme ponechali iba plánované dátumy.
  • Nevedeli sme ako spustiť MySQL na localhoste. Preto sme lokálne používali SQLite, hoci si uvedomujeme, že to nebolo dovolené. Hosting už ale používa MySQL databázu.
